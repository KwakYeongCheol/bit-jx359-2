메인 페이지
	1. 사용자는 로그인, 회원가입을 할 수 있다.
	2. 로그인한 사용자의 경우, 블로그를 설정할 수 있다.([블로그 - 생성]으로 이동)

회원 - 가입
	1. 아이디, 비밀번호, 이름을 입력한 후 회원가입 버튼을 누른다.
	2. 회원가입이 완료된 후 자동으로 로그인이 되고, [블로그 - 생성] 페이지로 이동한다.

	* 회원가입시 8자리 이상의 비밀번호를 생성해야한다.
	
회원 페이지
	1. 로그인한 사용자만 접근할 수 있다.
	2. 아이디, 이름을 볼 수 있다.

회원 - 수정
	1. 로그인한 사용자만 접근할 수 있다.
	2. 비밀번호를 변경할 수 있다.
	3. 수정한 비밀 번호를 저장한 후, [ 회원 페이지 ] 로 이동한다. 

블로그 - 생성
	1. 로그인한 사용자의 블로그가 존재하지 않을 경우
		1-1. 블로그 제목을 입력한 후 블로그를 생성할 수 있다.
		1-2. 기본적으로 "내 글" 이라는 기본 카테고리가 자동으로 생성된다.
	2. 블로그가 존재할 경우
		1-1. 블로그의 제목을 수정할 수 있다.
	3. 이 때 블로그 아이디는 로그인한 사용자의 아이디와 동일하다.


개인 블로그 - 메인 페이지
	1. 블로그에 게시된 글과 방명록을 볼 수 있다.
	2. 로그인한 유저가 블로그 주인일 경우, [관리자 페이지] 로 이동할 수 있다. 

	* 블로그가 존재하지 않으면 [메인 페이지] 로 이동한다. 

개인 블로그 - 카테고리
	1. 블로그에 카테고리의 해당하는 글들을 볼 수 있다.

개인 블로그 - 관리자 페이지
	1. 블로그의 주인만 접근할 수 있다.
	2. 모든 글을 확인할 수 있고, 작성, 수정, 삭제가 가능하다.
	3. 모든 방명록을 확인할 수 있고, 삭제가 가능하다.


개인 블로그 - 관리자 페이지 - 글 - 작성
	1. 블로그의 주인만 글을 작성할 수 있다.
	2. 카테고리를 선택하고, 글의 제목, 내용, 을 입력한 후, 작성 버튼을 누른다.
	3. 작성된 글을 저장한 후, [관리자 페이지]로 돌아간다.


개인 블로그 - 관리자 페이지 - 글 - 수정
	1. 블로그의 주인만 글을 수정할 수 있다.
	2. 글의 제목, 내용, 카테고리를 수정한 후 수정 버튼을 누른다.
	3. 작성된 글을 수정한 후, [관리자 페이지]로 돌아간다.


개인 블로그 - 관리자 페이지 - 글 - 삭제
	1. 블로그의 주인만 글을 삭제할 수 있다.
	2. 글을 삭제한 후, [관리자 페이지]로 돌아간다.


개인 블로그 - 관리자 페이지 - 카테고리 페이지
	1. 블로그의 주인만 접근할 수 있다.
	2. 내 블로그의 모든 카테고리를 확인, 추가, 수정, 삭제할 수 있다.

개인 블로그 - 관리자 페이지 - 카테고리 페이지 - 추가
	1. 블로그의 주인만 카테고리를 추가할 수 있다.
	2. 카테고리 제목을 입력한 후, 추가 버튼을 누른다.
	3. 카테고리가 저장된 후, [카테고리 페이지] 로 돌아간다.

개인 블로그 - 관리자 페이지 - 카테고리 페이지 - 수정
	1. 블로그의 주인만 카테고리를 수정할 수 있다.
	2. 카테고리 제목을 수정한 후, 수정 버튼을 누른다.
	3. 카테고리를 수정한 후, [카테고리 페이지] 로 돌아간다.

개인 블로그 - 관리자 페이지 - 카테고리 페이지 - 삭제
	1. 블로그의 주인만 카테고리를 삭제할 수 있다.
	2. 카테고리를 삭제한 후, [카테고리 페이지] 로 돌아간다.

	* 해당 카테고리에 글이 있을 경우, 삭제할 수 없다.
	* 해당 블로그의 하나 남은 카테고리라면 삭제할 수 없다. 


개인 블로그 - 방명록 페이지
	1. 블로그에 작성된 방명록을 볼 수 있다.
	2. 로그인한 유저의 경우, [방명록 - 작성] 을 할 수 있다.


개인 블로그 - 방명록 - 작성
	1. 로그인한 유저는 모든 블로그의 방명록을 작성할 수 있다.
	2. 방명록 내용을 작성한 후 작성 버튼을 누른다.
	3. 작성된 방명록을 저장한 후, [방명록 페이지] 로 돌아간다.

개인 블로그 - 방명록 - 수정
	1. 방명록을 작성한 유저만 해당 방명록을 수정할 수 있다.
	2. 방명록 내용을 수정한 후 수정 버튼을 누른다.
	3. 수정된 방명록을 저장한 후, [방명록 페이지] 로 돌아간다.

개인 블로그 - 방명록 - 삭제
	1. 블로그 주인과 방명록을 작성한 유저만 해당 방명록을 삭제할 수 있다.
	2. 방명록을 삭제한 후, [방명록 페이지] 로 돌아간다.
